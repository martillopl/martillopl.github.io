Tema 1. Tecnología básica — Cuadro comparativo

Este cuadro reúne las diferencias que más suelen confundirse en examen. La idea es que puedas repasar de un vistazo conceptos parecidos pero no equivalentes.

1. Dispositivos personales vs dispositivos corporativos

Aspecto Dispositivo personal / BYOD Dispositivo corporativo
Titularidad Es del usuario. Es de la organización.
Control técnico Más limitado. Más completo y centralizado.
Ventaja principal Mayor comodidad y movilidad. Mayor homogeneidad y control.
Problema principal Más riesgo para la privacidad y la seguridad. Mayor coste y necesidad de gestión interna.

2. Gestión remota vs asistencia remota

Aspecto Gestión remota Asistencia remota
Finalidad Administrar el dispositivo. Resolver una incidencia puntual.
Alcance Configuración, políticas, inventario, bloqueo, borrado. Soporte técnico temporal.
Uso típico Administración centralizada. Atención al usuario.
Idea clave Control permanente. Ayuda puntual.

3. Despliegue vs clonación

Aspecto Despliegue Clonación
Concepto Puesta en marcha estandarizada de equipos. Copia de una imagen o equipo maestro.
Relación Es el proceso general. Es una técnica concreta de despliegue.
Ventaja Rapidez y uniformidad. Reproducción exacta de una configuración.
Riesgo Errores de automatización. Duplicidades o conflictos de configuración.

4. Escritorio remoto vs escritorio virtual

Aspecto Escritorio remoto Escritorio virtual
Qué se accede A una sesión de un equipo remoto. A un entorno de trabajo alojado en infraestructura virtual.
Dependencia del hardware local La sesión está en otro equipo físico. El puesto depende menos del equipo local.
Uso típico Teletrabajo, soporte, acceso puntual. Centralización de puestos de trabajo.
Idea clave Se controla una sesión remota. Se consume un escritorio virtualizado.

5. Microsoft, Linux y macOS

Aspecto Microsoft Windows Linux macOS
Uso habitual Puesto de usuario. Servidores, infraestructura, técnica. Perfiles concretos y ecosistema Apple.
Implantación Muy extendida. Muy fuerte en sistemas y servidores. Menor presencia general.
Fortaleza Compatibilidad y entorno corporativo general. Flexibilidad y control técnico. Integración hardware-software.
Idea clave Entorno dominante en el puesto de usuario. Base de muchos servicios técnicos. Alternativa integrada para ciertos perfiles.

6. Software libre vs código abierto

Aspecto Software libre Código abierto
Enfoque principal Libertades del usuario. Acceso al código y modelo colaborativo.
Qué destaca Usar, estudiar, modificar y redistribuir. Transparencia y desarrollo abierto.
Relación entre ambos Muy próximos, pero no idénticos. Comparten buena parte de sus principios prácticos.
Idea clave La libertad del usuario. La apertura del código.

7. Licencias permisivas vs copyleft

Aspecto Licencia permisiva Licencia copyleft
Restricciones Pocas. Más exigentes con las obras derivadas.
Uso en proyectos propios Muy flexible. Permitido, pero con condiciones de continuidad.
Obras derivadas Pueden licenciarse con mayor libertad. Deben mantenerse bajo la misma lógica de licencia.
Idea clave Libertad de reutilización. Preservar la apertura del software derivado.

8. Virtualización de escritorios vs virtualización de aplicaciones

Aspecto Virtualización de escritorios Virtualización de aplicaciones
Qué se virtualiza Todo el entorno de escritorio. Solo la aplicación.
Alcance Más amplio. Más específico.
Uso típico Entornos de trabajo completos. Entrega aislada de aplicaciones concretas.
Idea clave Virtualiza el puesto entero. Virtualiza solo el programa.

9. Nube pública, privada e híbrida

Aspecto Nube pública Nube privada Nube híbrida
Infraestructura Compartida por varios clientes. Dedicada a una sola organización. Combina ambos modelos.
Control Menor. Mayor. Intermedio y distribuido.
Flexibilidad Muy alta. Más orientada al control. Alta, pero con complejidad.
Idea clave Elasticidad. Control. Equilibrio entre ambas.

Resumen mental rápido: BYOD = dispositivo del usuario; gestión remota = control a distancia; clonación = copia de equipos; escritorio remoto = acceso a sesión; escritorio virtual = entorno centralizado; pública = compartida; privada = dedicada; híbrida = mezcla.
