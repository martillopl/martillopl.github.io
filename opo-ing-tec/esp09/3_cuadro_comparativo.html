Proyecto básico vs. Proyecto de ejecución

Confusión frecuente en test: el proyecto básico es suficiente para solicitar la licencia urbanística pero NO para iniciar las obras. El proyecto de ejecución es el documento de referencia para construir y para la dirección de obra.
AspectoProyecto básicoProyecto de ejecución
¿Qué contiene?Las características generales del edificio y sus prestaciones, mediante la adopción y justificación de las soluciones generales básicas. Define tipología y organización del edificioEl desarrollo completo del proyecto básico: todos los detalles constructivos, cálculos estructurales, especificaciones de instalaciones, pliegos de condiciones completos
¿Para qué sirve?Para solicitar la licencia urbanística de obras (en muchos Ayuntamientos es suficiente)Para iniciar y dirigir la construcción del edificio
¿Permite iniciar las obras?NO — no contiene el detalle necesario para construirSÍ — es el documento de referencia para la construcción
¿Quién debe firmarlo?El proyectista competente para el tipo de obra (arquitecto, ingeniero, etc.)El mismo proyectista o proyectistas que firmaron el proyecto básico
Contenido documentalMemoria descriptiva y justificativa (simplificada) · planos de planta, alzados y secciones · memoria de calidadesMemoria completa + justificación CTE + cálculos estructurales · planos completos (incluidos detalles constructivos e instalaciones) · pliego de condiciones completo · mediciones y presupuesto
